# Prompt Manager 隱私權政策

**最後更新日期：** 2025 年 12 月 31 日

**Prompt Manager**（以下簡稱「我們」）致力於保護您的隱私。本隱私權政策旨在說明我們的 Chrome 擴充功能如何運作、如何處理您的資料，以及我們請求特定權限的理由。

## 核心原則：本地優先 (Local-First) 與隱私至上

Prompt Manager 採用 **「本地優先」** 的架構設計。這意味著：

1. **無雲端同步**：我們不營運任何後端伺服器、資料庫或雲端儲存空間。
2. **無追蹤分析**：我們未內建任何追蹤代碼（如 Google Analytics、Mixpanel）來監控您的使用行為。
3. **不收集資料**：我們不會收集、儲存、分享或販售您的個人資料、Prompt 內容或瀏覽紀錄。

您在擴充功能中建立的所有資料，僅會儲存在您的本地裝置上。

## 1. 資料儲存

### 用戶生成內容

您的 Prompt、變數、設定與標籤皆透過 `chrome.storage.local` API 儲存於您的瀏覽器中。這些資料受 Chrome 安全沙箱保護，我們或任何第三方皆無法存取。

### 資料備份

由於我們不將資料儲存於伺服器，您需自行負責資料備份。我們在設定中提供了「匯出 JSON」功能，讓您可以將資料備份至您的電腦硬碟。

## 2. 權限使用說明

為了提供完整功能，Prompt Manager 需要請求特定權限。我們嚴格遵守 **「最小權限原則」**：

- **`activeTab` (當前分頁存取)**：
  - **用途**：讓「Spotlight 懸浮視窗」（透過 `Alt+P` 或點擊圖示開啟）能顯示於您當前瀏覽的網頁之上，並在您使用「頁面元素」變數時，協助獲取頁面資訊（如標題、網址或 HTML 快照）。
  - **運作方式**：此權限僅在您**主動呼叫**擴充功能時，暫時授予我們對當前分頁的存取權。我們無法透過此權限在背景監控您的瀏覽紀錄。
- **`scripting` (腳本執行)**：
  - **用途**：用於確認文字插入程式 (`content.js`) 是否已載入、執行插入動作，以及獲取「頁面元素」變數的資料。
  - **運作方式**：當您使用「插入」功能時，擴充功能會將您的 Prompt 文字填入網頁的輸入框中。若您使用「進階變數」，我們可能會執行腳本以抓取您指定的網頁內容（如 `h1`）。所有用戶自定義的腳本皆在受限制的 **Sandbox 安全沙箱** 中執行，確保安全性。
- **`sidePanel` (側邊欄)**：
  - **用途**：允許擴充功能在瀏覽器的側邊欄顯示管理介面。
- **`storage` (儲存)**：
  - **用途**：將您的 Prompt 與設定儲存在您的本地裝置中。

## 3. 與網頁的互動

本擴充功能僅在以下嚴格條件下與網頁（DOM）進行互動：

1. **Spotlight 懸浮視窗**：當您按下快捷鍵或點擊圖示時，我們會在頁面中注入暫時性的介面 (`spotlight.js`)，供您搜尋與選擇 Prompt。
2. **文字插入**：當您確認執行操作時，我們會將文字填入您當前聚焦的輸入框。

本擴充功能**不會**被動地讀取、記錄或傳輸您所訪問網站的任何內容。

## 4. 政策更新

我們可能會不時更新本隱私權政策，以反映我們操作方式的變更，或出於其他營運、法律或監管原因。如有重大變更，我們將更新本政策頂部的「最後更新日期」。
